COMMUNICATION GRAPHICS

Narrative

This course was an exploration of visual communication and layout design using Adobe Photoshop and Illustrator. Over the course of the semester key exercises were undertaken — designing an editorial magazine cover and redesigning a movie poster using only typography. Each exercise built an understanding of composition, visual hierarchy, typography and the fundamentals of graphic design thinking.
