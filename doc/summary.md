# Project Summary — Alma Rachel Ipe Portfolio

## Overview

This is a personal portfolio website for **Alma Rachel Ipe**, a third-year Textile and Apparel Design student at the National Institute of Design, Andhra Pradesh. The site showcases her work across textile design, surface development, apparel construction, and visual communication.

## Site Structure

```
index.html          — Single-page homepage (hero video slideshow, project grid, about, contact)
style.css           — Shared styles for project detail pages
projects/           — Individual project detail pages
images/             — Project imagery
videos/             — Hero section video files
```

## Project Categories

The work is organised into five sections:

1. **Woven** — Handloom weaving explorations including basic weave structures (plain, twill, satin), advanced techniques (double weave, supplementary weft), and a full woven design project.
2. **Print** — Screen printing and digital print work covering repeat patterns, motif development, and concept-driven surface design.
3. **Surface Development** — Non-woven fabric construction (felting, bonding, layering) and hand embroidery as a surface practice.
4. **Apparel** — Garment construction using self-developed textiles, where fabric properties inform silhouette and construction.
5. **Narrative** — Photography documenting craft and material processes, plus graphic design and visual communication work.

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript — no frameworks or build tools.
- Google Fonts: Cormorant Garamond (serif) and Inter (sans-serif).
- Mobile-responsive layout with a hamburger menu at the 768px breakpoint.
- Fullscreen video slideshow on the homepage hero section with auto-advancement.
